# Terms — глоссарий терминов RSpace

Используй эти термины единообразно во всей документации. Если появился новый — сначала добавь сюда, потом используй.

## Финансовые термины

### Баллы

Внутренняя валюта RSpace (`1 балл = 1 рубль`). Пополняется через банковскую карту (CloudPayments). Тратится на: подписку, услуги (юристы, проверки, ДКП сверх лимита), продвижение на классифайдах. **Не выводится** на банковский счёт.

### Рубли (баланс вывода)

Доход риелтора — агентские комиссии от банков-партнёров, страховых, застройщиков. **Выводится** на банковский счёт, или конвертируется в баллы для новых услуг внутри RSpace.

### Агентская комиссия

Процент, который **банк / страховая / застройщик** платит RSpace за привлечённого клиента. Часть этой комиссии получает **риелтор** (на рублёвый счёт), часть остаётся платформе. Конкретная формула зависит от тарифа — чем выше тариф, тем больший % риелтору. Точные цифры см. `knowledge/data-points.md`.

**Пишем «агентская комиссия», НЕ «кэшбек».** «Кэшбек» подразумевает скидку клиенту — это не то, что делает RSpace.

### Service Fee

Синоним «агентской комиссии риелтору» в внутренних документах. Процент зависит от тарифа: Триал 30% / Профи 25% / Премиум 19% / Ультима 15% (числа из xlsx-модели, по какой базе считается — **уточняется у команды**). В публичной документации термин **«Service Fee» не используем** — пишем «агентская комиссия».

### CloudPayments

Платёжный провайдер для пополнения баллов банковской картой. Поддерживает 3-D Secure. В публичной документации упоминаем как контекст (при объяснении холда 1 ₽ при триале), не как инструмент выбора.

## Тарифы

Пять уровней:

- **Триал** — 30 дней бесплатно. Требует привязки карты. AI-юриста нет.
- **Профи** — базовый тариф. 3 объекта. AI-юриста нет. −20% на услуги.
- **Премиум** — ключевой апсейл. 5 объектов. **AI-юрист Грут включён**. −25%.
- **Ультима** — топ для активных агентов. 10 объектов. AI-юрист. −30%.
- **Энтерпрайс** — для портфеля 10+. 15 объектов. AI-юрист. −30%. Единая цена столицы/регионы.

Пишем с заглавной: «Профи», «Премиум», «Ультима», «Энтерпрайс», «Триал».

## Услуги

### Юрист на сделку

Живой юрист сопровождает сделку от подготовки ДКП до регистрации. Платно, со скидкой по тарифу.

### Проверка объекта

Юридическая проверка недвижимости — **3 пункта**: Отчёт ЕГРН, Проверка дома на аварийность, История владения. См. `_sources/check-types-breakdown.md`.

### Проверка собственника

Проверка физ.лица — **8 пунктов**: Банкротство, ФССП, Паспорта, Доверенности, Реестр иноагентов, Розыск, Росфинмониторинг, Коммерческая деятельность.

### Комплексная проверка (объект + собственник)

Union двух предыдущих — **11 пунктов**.

### Подготовка ДКП

Юрист готовит договор купли-продажи по вашим данным. По лимиту тарифа — бесплатно (Профи 1, Премиум 2, Ультима/Энтерпрайс 3 в месяц). Сверх лимита — платно со скидкой.

### AI-юрист Грут

GPT-бот для быстрых юридических вопросов. Модель `o4-mini-2025-04-16` с уровнем мышления «Глубоко». **Доступен с Премиума** (Премиум / Ультима / Энтерпрайс). **Не** доступен на Триале и Профи.

Пишется через дефис: «AI-юрист Грут».

## Кабинет

### ЛК (Личный кабинет)

`lk.rspace.pro` — основной UI для риелтора. **Tech:** React Router 7 SSR + Chakra UI v3 + TanStack Query + Jotai. Backend — общий с админкой.

### Админка

`admin.rspace.pro` — внутренний UI команды RSpace (биллинг, поддержка, разработчики). Tech-стек и источник кода — TBD (уточнить у команды).

### Лендинг

`rspace.pro` — маркетинговая страница. Next.js 15.4 + React 19 + Tailwind v4 + Radix UI. Репо: `git.rs-app.ru/rspase/landing/next`.

## Лиды

### Источники

1. **Авито (звонки и сообщения)** — через API, pull каждые ~15 минут.
2. **ЦИАН (звонки и сообщения)** — через API, pull ~15 минут.
3. **Лиды от застройщиков** — выгружаются через общую форму (Yandex.Диск / Google Spreadsheets). Домен сайта застройщика = источник (`strana-siti.ru`, `soul-forma.ru` и т.п.).
4. **JivoSite чат** (на сайте RSpace) — webhook мгновенно.

### Статусы (сейчас в продукте)

- **Доступен (`available`)** — лид в общем пуле, не назначен.
- **Назначен (`applied`)** — привязан к конкретному риелтору.

Более сложный workflow (связался / встреча / сделка) — ведётся во внешней CRM, если риелтор подключил (но в публичной документации об этом **не упоминаем** до отдельного решения команды).

## Биллинг

### Триал

30 дней. Требует привязки банковской карты через CloudPayments (3DS + холд 1 ₽). Списаний в течение триала нет. Карту можно отвязать сразу после активации — триал продолжит работать.

### Автосписание

По окончании триала — автосписание с карты за первый месяц. Далее каждые 30 дней списание с **баллов** (не с карты). Если баллов не хватает — нужно пополнить.

### Отмена подписки

В кабинете → «Подписка» → «Отменить». Автосписание отключается, тариф работает до конца оплаченного периода.
