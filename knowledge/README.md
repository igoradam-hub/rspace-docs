# /knowledge/ — база знаний для AI-сессий

Эта папка — справочная база для любого Claude (или другого AI/разработчика), работающего над rspace-docs. Источник истины для стилистики, терминов, актуальных цифр и workflow.

## Файлы

| Файл | Что там | Когда читать |
|---|---|---|
| [`voice.md`](./voice.md) | Стилистика публичной документации: тон, эмодзи (нет), имена (нет), метастроки (нет) | Перед любой правкой `external/*.md` |
| [`terms.md`](./terms.md) | Глоссарий — какие термины пишем, какие не пишем, почему «агентская комиссия» а не «кэшбек» | При сомнениях в терминологии |
| [`data-points.md`](./data-points.md) | Актуальные цифры: тарифы, цены услуг, лимиты, контакты. Source of truth `_sources/*.xlsx` | При обновлении цифр в тексте |
| [`workflow.md`](./workflow.md) | Как пушить, как синкать GitBook, как снимать скрины, как чинить если не работает | При работе с инфраструктурой |

## Корневой `CLAUDE.md`

В корне репо лежит [`CLAUDE.md`](../CLAUDE.md) — **главный** файл для AI-сессий. Начинай **всегда** с него. Этот README и файлы `knowledge/` — детализация для узких тем.

## Порядок чтения в новой сессии

1. **`CLAUDE.md`** (корень) — понять проект и критичные правила.
2. **`knowledge/voice.md`** — если правишь публичную документацию.
3. **`knowledge/terms.md`** — если в правках новые термины.
4. **`knowledge/data-points.md`** — если правишь цифры / контакты.
5. **`knowledge/workflow.md`** — если настраиваешь что-то в инфраструктуре.
6. **`CHANGELOG.md`** (корень) — контекст что было сделано ранее.
7. **`_verification/DISCREPANCY-DECISIONS-2026-04-23.md`** — открытые вопросы к команде.

## Как обновлять эту базу

Если во время работы:

- Появился **новый термин** → добавь в `terms.md`
- Изменилась **цифра** (цена, лимит, контакт) → обнови `data-points.md`
- Появился **новый workflow или инструмент** → допиши в `workflow.md`
- Изменился **подход к стилистике** → правь `voice.md` и обновляй `CHANGELOG.md`

База должна быть **живой**. Если в старой сессии Claude что-то решил, но не записал — при следующей сессии он забудет. **Записывай.**
