## Продукт

**RSpace** (домены: [rspace.pro](https://rspace.pro), [lk.rspace.pro](https://lk.rspace.pro), [admin.rspace.pro](https://admin.rspace.pro)) — B2B SaaS для независимых риелторов в РФ. Единая платформа, которая заменяет ~5 разрозненных сервисов: мультилистинг на классифайдах (Авито, ЦИАН, ДомКлик), юридические услуги (ДКП, проверки, сопровождение), ипотечный брокер, страховой брокер, CRM-заготовка.

- **Стадия:** Pre-seed, MVP в проде, ищем product-market fit.
- **Основан:** 2025.
- **CEO:** Алексей Козлов.
- **CPO:** Игорь Адам.

## Глоссарий

### Продуктовые термины

| Термин | Значение |
|---|---|
| **Классифайд / площадка** | Внешняя платформа для публикации объявлений: Авито, ЦИАН, ДомКлик, Яндекс.Недвижимость |
| **Публикация / выкладка** | Размещение объекта на классифайдах |
| **Лид** | Контакт клиента, пришедший с площадки (через звонок, форму, Jivo-чат) |
| **Объект / listing** | Квартира, дом или участок в рекламе агента |
| **Сделка** | Завершение купли-продажи объекта |
| **ДКП** | Договор купли-продажи |
| **ДДУ** | Договор долевого участия (для новостроек) |
| **Скоринг** | Юридическая проверка (собственника, объекта, ипотеки) — платная услуга |
| **Триал** | Пробный 30-дневный период |
| **Клубные цены** | Сниженные цены на услуги для подписчиков; скидка зависит от тарифа (−20 / −25 / −30%) |
| **Вторичка** | Вторичный рынок недвижимости |
| **Первичка** | Новостройки |
| **Public offer** | Персональная страница объекта для клиента (с лого и цветами агента) |

### Технические термины

| Термин | Значение |
|---|---|
| **Backend** | Laravel 13 API, репо `project/backend` |
| **Frontend** | React Router 7 SSR-приложение кабинета агента, репо `project/frontend` |
| **Landing** | Next.js 15 маркетинговый сайт, репо `landing/next` |
| **lk.rspace.pro** | Домен кабинета |
| **api.rspace.pro** | Домен backend API |
| **admin.rspace.pro** | Домен админки (UI — отдельный репо вне GitLab, backend общий) |
| **Feed** | Статический XML/JSON для синхронизации с ДомКлик, Авито (outbound) |
| **Sanctum-токен** | Bearer-токен Laravel Sanctum для авторизации API-запросов |
| **CP** | CloudPayments (платёжный провайдер) |
| **AmoCRM** | Внешняя CRM, куда уходят лиды (outbound webhook) |
