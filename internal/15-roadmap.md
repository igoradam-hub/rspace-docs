> **Аудитория:** продукт, команда, инвесторы (краткая версия наружу).

## Позиционирование

> **«RSpace — инфраструктура агентства для тех, кто работает на себя. 0% комиссии, все инструменты в одном месте.»**

## 4 критических гэпа (из CEO-спичей 9 марта)

1. **Zero notifications** — 18 мин/неделю = агент заходит и забывает.
   - **Решение:** Telegram-бот + email-дайджест еженедельный.
   - **Статус:** в работе.

2. **Dead referral program** — CAC=0 канал простаивает.
   - **Решение:** сделать UI в кабинете, показать reward.
   - **Статус:** не начата.

3. **PostHog blind spot** — не видим поведение на фронте.
   - **Решение:** активировать провайдер в Layout.
   - **Статус:** в работе.

4. **Bus factor на Юлю** — она R в 5 функциях.
   - **Решение:** документирование процессов + передача Маше.
   - **Статус:** обсуждается, не формализовано.

## Стратегия роста

### Каналы привлечения (приоритет)
1. **Рефералы** — CAC=0, качественный трафик. Нужен UI.
2. **Посевы в Telegram** — экспериментально (см. `RSpace_Посевы_Аналитика.xlsx`).
3. **Яндекс.Директ / Я.Дзен** — платный трафик с UTM.
4. **Контент-маркетинг** — Даниил, рабочий канал Дзен / блог.
5. **Партнёрские интеграции** — застройщики (клубные цены), банки (ипотека).

### Удержание
1. **Активные уведомления** — Telegram каждое событие.
2. **Еженедельная сводка** — email-дайджест о работе агента.
3. **Возврат потерянных** — email-серия для 186 юзеров без подписки.
4. **Качество поддержки** — Telegram-чат с быстрыми ответами.

### Монетизация
1. **Подписки** — фикс (Профи / Премиум / Ультима / Энтерпрайс).
2. **Услуги** — per-use с скидкой по тарифу.
3. **Ипотечные комиссии** — процент от сделки.
4. **Страховые комиссии** — процент.
5. **Клубные цены** — комиссия с застройщиков.

## Команда (запланированный рост)

Текущая команда — ~7 человек.

Для масштабирования до 100 MRR / 1K платящих потребуется:
- +2 разработчика (backend + frontend).
- +1 QA.
- +1 DevOps / Platform engineer.
- +2 менеджера (продажи / онбординг).
- +1 юрист (full-time на услугах).

## Зависимости и риски

### Критические зависимости
- **CloudPayments** — если упадёт, платежи встают. Нужен fallback (Tinkoff, YooKassa).
- **Avito / ЦИАН API** — если отключат, продукт теряет ядро. Мониторить контракты.
- **sms.ru** — если упадёт, регистрация ломается. Fallback SMSC.
- **Ключевые люди** (Артём, Юля, Света) — bus factor.

### Стратегические риски
- **Конкуренты** — если Этажи / ЦИАН запустят аналог для независимых → цена подписки упадёт.
- **Регуляторы** — 152-ФЗ, самозанятые, изменения в налогах.
- **Банки сократят комиссии** — ипотечный кэшбек 1.5% — ключевой revenue для юзера. Если упадёт до 0.5%, мотивация юзера теряется.
