# AI-генерация описаний объектов: промпты, модель, план

---

## 1. SYSTEM PROMPT (вставить в админку)

```
Ты - копирайтер-риелтор. Пишешь описания объектов недвижимости для публикации на Авито, ЦИАН, ДомКлик.

СТИЛЬ:
- Пиши от третьего лица, без обращений к читателю
- Конкретика и факты, не абстрактные эпитеты. Не "прекрасный вид" а "окна выходят на парк / во двор / на юг"
- Короткие предложения. Абзацы по 2-3 предложения
- Длина: 800-1200 символов. Не длиннее
- Начинай сразу с главного преимущества объекта, не с общих фраз

СТРУКТУРА ОПИСАНИЯ:
1. Первое предложение - зацепка: главное преимущество (локация, ремонт, планировка, цена ниже рынка)
2. Расположение и район: что рядом, транспорт, инфраструктура. Конкретные расстояния в минутах
3. Характеристики объекта: планировка, площади, этаж, состояние, особенности
4. Ремонт и отделка: если есть данные, опиши что сделано
5. Дополнительно: парковка, вид, тихий двор, консьерж - если есть в данных
6. Закрывающее предложение: для кого подойдет объект (семья, инвестиция, первое жилье)

ЗАПРЕЩЕНО:
- Телефоны, email, ссылки, контактные данные
- "Торг уместен", "цена договорная", "звоните"
- КАПСЛОК, !!! и ???
- Эмодзи
- "Срочно", "скидка", "акция", "только сегодня"
- Вранье и приукрашивание: не добавляй того, чего нет в данных
- Фразы "не упустите шанс", "уникальное предложение", "эксклюзив"
- Слово "данный" (данная квартира, данный объект)
- Обращения: "вы", "вам", "позвоните нам"
- Оценочные прилагательные без основания: "шикарный", "невероятный", "потрясающий"

ПРАВИЛА ПЛОЩАДОК:
- Текст должен быть уникальным (не шаблонный копипаст)
- Не упоминай конкурирующие площадки (не пиши "как на Авито" или "смотрите на ЦИАН")
- Не указывай цену в описании (она в отдельном поле)
- Не дублируй то, что уже есть в структурированных полях (площадь, этаж, комнаты) - упоминай их в контексте, а не просто перечисляй

ТОНАЛЬНОСТЬ:
- Спокойная, уверенная, профессиональная
- Как будто опытный агент рассказывает про объект знакомому
- Без восторженности, без давления
```

## 2. USER PROMPT TEMPLATE (вставить в админку)

```
Напиши описание для объявления о недвижимости на основе следующих данных:

Тип: {type}
Сделка: {deal_type}
Город: {city}
Район: {district}
Адрес: {street}
Площадь общая: {area_total} м²
Площадь жилая: {area_living} м²
Площадь кухни: {area_kitchen} м²
Комнат: {rooms}
Этаж: {floor} из {floors_total}
Цена: {price} ₽
Год постройки: {year_built}
Ремонт: {renovation}
Балкон: {balcony}
Санузел: {bathroom}
Тип дома: {house_type}

Дополнительно (если заполнено):
{all_fields_json}

Правила:
- Используй ТОЛЬКО те данные, которые предоставлены выше. Не выдумывай
- Если поле пустое или отсутствует - не упоминай его
- Длина: 800-1200 символов
- Пиши на русском языке
- Не дублируй данные из полей - раскрывай их в контексте
```

## 3. РЕКОМЕНДАЦИЯ ПО МОДЕЛИ

### Для MVP (старт):

**GPT-4o-mini** через OpenAI API

Почему:
- Стоимость: ~$0.15 на 1M input / $0.60 на 1M output. Одна генерация = ~$0.001-0.002 (меньше 0.2 рубля)
- Скорость: 3-5 секунд на генерацию (в пределах 30с таймаута)
- Качество: для структурированного текста по шаблону - более чем достаточно
- При 100 генерациях в день = ~$6/месяц

Почему НЕ GPT-4o:
- В 15-30 раз дороже
- Для описания квартиры по шаблону - оверкилл
- Медленнее (7-15 секунд)

### Параметры для админки:

| Параметр | Значение | Почему |
|----------|----------|--------|
| API URL | `https://api.openai.com/v1/chat/completions` | Стандартный OpenAI |
| Model | `gpt-4o-mini` | Баланс цена/качество |
| Max tokens | `800` | ~1200 символов русского текста |
| Temperature | `0.7` | Разнообразие без бреда. Если тексты слишком однообразные - поднять до 0.8. Если галлюцинирует - опустить до 0.5 |

### Альтернативы (если OpenAI не подходит):

1. **Claude 3.5 Haiku** (Anthropic) - дешевый, быстрый, хорошо пишет по-русски. API совместим с OpenAI форматом через адаптер
2. **Gemini 2.0 Flash** (Google) - бесплатный тир до 15 RPM, потом дешевый. Русский норм
3. **YandexGPT** - если нужен российский провайдер. API другой формат, потребует адаптер на бэке

### Путь масштабирования:

```
MVP (март):     GPT-4o-mini + хороший промпт
Итерация 1:     Few-shot примеры в system prompt (5 лучших описаний)
Итерация 2:     A/B тест разных температур и промптов (руками через админку)
После раунда:   RAG - подтягивать похожие топовые объявления из базы
Когда-нибудь:   Файнтюн на своих данных (нужно 500+ примеров)
```

## 4. FEW-SHOT ПРИМЕРЫ (добавить в system prompt после первых тестов)

Когда будешь итерировать промпт, добавь в конец system prompt блок с примерами. Вот шаблон:

```
ПРИМЕРЫ ХОРОШИХ ОПИСАНИЙ:

Пример 1 (квартира, продажа):
---
Тихий центр Нижнего Новгорода, 5 минут пешком до метро Горьковская. Двухкомнатная квартира на 7 этаже 17-этажного монолитного дома 2019 года постройки.

Планировка европейская: кухня-гостиная 18 м² и изолированная спальня 14 м². Общая площадь 52 м². Потолки 2.8 м. Два больших окна на юго-запад, вечером много света.

Свежий ремонт: ламинат, обои под покраску, встроенная кухня с техникой остается. Раздельный санузел с итальянской плиткой. Застекленная лоджия 4 м² с видом во двор.

Закрытая территория, подземная парковка (место включено в стоимость), детская площадка. Рядом школа №45, Сормовский парк, Ашан.

Подойдет молодой семье или для сдачи в аренду - район с высоким спросом.
---

Пример 2 (дом, продажа):
---
Готовый к проживанию дом 180 м² на участке 8 соток в поселке Новинки, 15 км от центра Нижнего Новгорода по Богородской трассе.

Два этажа: на первом кухня-столовая 25 м², гостиная с камином, гостевой санузел. На втором три спальни и полноценная ванная. Потолки 3 метра. Панорамные окна в гостиной.

Дом из газобетона, утепление 150 мм, облицовка клинкерным кирпичом. Отопление газовое (магистральный газ подключен). Центральная канализация. Скважина 42 м.

Участок ровный, огорожен, ландшафтный дизайн, зона барбекю, навес на 2 машины. Рядом сосновый лес и Ока.

Для семьи, которая хочет жить за городом без компромиссов по комфорту.
---
```

Замени эти примеры на РЕАЛЬНЫЕ описания с Авито/ЦИАН которые показали хорошую конверсию (много просмотров, быстрая продажа). Чем ближе примеры к твоей аудитории (регионы, частные агенты) - тем лучше.

## 5. ЧЕКЛИСТ РЕСЕРЧА КОНВЕРСИОННЫХ ОБЪЯВЛЕНИЙ

Это твоя часть работы. Вот конкретный план:

### 5.1. Сбор примеров (2-3 часа)

- [ ] Открой Авито Недвижимость, выбери 3 города (Нижний, Казань, Уфа - регионы вашей аудитории)
- [ ] Для каждого города найди 10 объявлений с максимальным количеством просмотров
- [ ] Скопируй описания в Google Doc / Buildin.ai
- [ ] То же самое на ЦИАН: 10 топовых по просмотрам
- [ ] То же самое на ДомКлик: 10 топовых
- [ ] Итого: ~90 описаний

### 5.2. Анализ паттернов (1-2 часа)

Для каждого описания отметь:
- [ ] Длина (символы)
- [ ] Структура (с чего начинается, чем заканчивается)
- [ ] Использует ли списки или сплошной текст
- [ ] Упоминает ли инфраструктуру
- [ ] Есть ли "для кого подойдет"
- [ ] Тональность (сухая/теплая/продающая)

### 5.3. Антипаттерны (что у плохих объявлений)

- [ ] Найди 10 объявлений с минимальными просмотрами
- [ ] Выпиши что у них общего (обычно: короткие, без структуры, копипаст характеристик)

### 5.4. Правила модерации площадок

- [ ] Прочитай правила Авито для недвижимости: https://ads-help.avito.com/moderation
- [ ] Прочитай правила ЦИАН
- [ ] Выпиши запрещенные слова/фразы - добавить в ЗАПРЕЩЕНО в system prompt

### 5.5. Тестирование промптов (после выкатки инфраструктуры)

- [ ] Создай 5 тестовых объектов разных типов (квартира 1к, квартира 3к, дом, участок, комната)
- [ ] Сгенерируй описание для каждого
- [ ] Оцени по критериям: уникальность, конкретика, длина, тональность, пройдет ли модерацию
- [ ] Покажи Юле - она знает что реально работает
- [ ] Покажи 2-3 реальным агентам - их реакция важнее твоей
- [ ] Итерируй промпт: 3-5 раундов правок

## 6. АНТИПАТТЕРНЫ AI-ТЕКСТА (что точно нужно отловить)

Модели любят генерить определенные штампы. Добавь в system prompt запрет на:

- "Идеальный вариант для..." (слишком продающе)
- "Не упустите возможность..." (давление)
- "Обратите внимание на..." (обращение)
- "Данная квартира..." (канцелярит)
- "Гармонично сочетается..." (AI-штамп)
- "Продуманная планировка..." (если нет данных чем именно продумана)
- "Уютная атмосфера..." (пустое)
- Любые метафоры: "оазис тишины", "жемчужина района"

По мере тестирования будешь находить новые - добавляй в раздел ЗАПРЕЩЕНО.

## 7. ФОРМАТ ЗАПРОСА К API (для разработки)

Вот как будет выглядеть запрос от бэкенда:

```json
{
  "model": "gpt-4o-mini",
  "messages": [
    {
      "role": "system",
      "content": "<system_prompt из админки>"
    },
    {
      "role": "user",
      "content": "<user_prompt_template с подставленными данными>"
    }
  ],
  "max_tokens": 800,
  "temperature": 0.7
}
```

Ответ:
```json
{
  "choices": [
    {
      "message": {
        "content": "Тихий центр Нижнего Новгорода..."
      }
    }
  ]
}
```

Бэкенд берет `choices[0].message.content` и отдает фронту.

## 8. МЕТРИКИ УСПЕХА (как понять что работает)

На MVP без встроенной аналитики - считай руками:

- **Adoption rate**: сколько % пользователей нажали кнопку (смотри в PostHog, если починишь)
- **Accept rate**: сколько % оставили сгенерированный текст (vs удалили/переписали)
- **Качество**: попроси 5 агентов оценить 10 сгенерированных описаний по 5-балльной шкале
- **Модерация**: сколько объявлений с AI-описанием прошли модерацию с первого раза

## 9. СТОИМОСТЬ (для Лёши)

При GPT-4o-mini:
- 1 генерация ≈ 0.15 рубля
- 100 генераций/день ≈ 15 руб/день ≈ 450 руб/месяц
- 1000 генераций/день ≈ 150 руб/день ≈ 4500 руб/месяц

Это копейки. Даже при 10x росте - меньше 5К/месяц.
